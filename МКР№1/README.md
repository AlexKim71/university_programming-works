
за 25 апреля модульная работа 2025 год






Дано натуральне число N і одновимірний масив A1, A2, …, AN символьних елементів. Виконати сортування елементів послідовності за зростанням.
Програмний код повинен бути написаний з застосуванням структурного програмування, в завданні повинні бути виділені окремі підзадачі, кожна підзадача реалізована функцією користувача. Результат роботи програми записується в файл.
Немає обмежень на типи даних, структури даних, типи файлів, що використовує студент при написанні програмного коду.
