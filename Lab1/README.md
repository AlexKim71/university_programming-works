
Лабораторна робота 1. Ознайомлення з роботою в інтегрованому середовищі розробки. Розробка програм лінійної структури
Початок приймання: Monday 4 September 2023 08:00 AM
Термін спливає: Friday 13 September 2024 18:00 PM

Мета роботи: отримання практичних навичок роботи з програмним середовищем, командами меню, вікнами, піктограмами. Вивчення засобів програмування для створення програм лінійної структури.

Завдання для підготовки до роботи:

1. Ознайомитись з принципами розробки програм в середовищі програмування за вибором.

2. Вивчити структуру програм, числові типи даних, правила запису арифметичних виразів, організацію стандартного введення-виведення даних.

3. Розробити алгоритм та програму для вирішення завдань за формулами для відповідного варіанта з таблиці. Визначити область припустимих значень змінних. Вихідні дані вводяться з клавіатури, результати повинні бути відображені на екрані.

4. Підібрати тестові дані.

5. При захисті звіту необхідно відповідати на контрольні запитання та вміти пояснювати роботу програми.
